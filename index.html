Summer Projects

About:

We will firstly be work on the controls of motors using microcontrollers. Then jump to Robotics with use of Inverse Kinematics for arm movement In the end: The arm wil be able to pick and place a randomly placed object autonomously

Project 3 : Autonomous Navigation of Mars Rover.

About:

To design and simulate the system of autonomous navigation for the mars rover which mainly covers up obstacle detection and classification, obstacle avoidance and path planning to a specific location

Project 4 : Gyrobot:Self Balancing Robot

About:

A two-wheeled self-balancing robot that achieves perfect equilibrium, using technology and precision engineering. The robot has locomotion capabilities while maintaining its balance similar to humans
